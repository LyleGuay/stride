Tradeoffs summary
A — Stacked: Most readable, most chrome. Two rows above the date nav = three rows total on Daily tab.
B — Name + pills: Compact single row. Pills feel slightly inconsistent next to the desktop underline style.
C — Name as label: Elegant, single row. "Calorie Log" gets too long — would need abbreviation or truncation.
D — Muted name: Minimal change to current layout. Name is clearly secondary to tabs. Works well with long names.
