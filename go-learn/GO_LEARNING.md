# Format Verbs:
Common Format Verbs

%s - string
%d - integer (decimal)
%f - float (default precision)
%.2f - float with 2 decimal places
%t - boolean
%v - default format for any value
%+v - struct with field names
%#v - Go syntax representation
%T - type of value
%p - pointer address
